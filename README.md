# WATERMeDIA BINARIES
WaterMedia Binaries is a collection of precompiled binaries for various tools and 
libraries used in multimedia decoding such as FFMPEG or libVLC.

Ships binaries for all platforms (Windows, Linux, MacOS) and architectures (x64, ARM64).<br>
NOTE: 32-bit architectures are not supported.

# Shipped binaries
- FFMPEG
- libVLC
- ISPCTextureCompressor (Samsung fork)
- Sevenzipjbinding
